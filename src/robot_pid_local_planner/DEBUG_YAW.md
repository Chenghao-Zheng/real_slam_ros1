## 终点朝向控制 - 快速调试指南

### 现在的设置（已启用）

```yaml
ENABLE_FINAL_YAW: true       # ✅ 已启用
YAW_TOL: 0.1                 # 容差 ≈ 5.7°
YAW_ADJUST_TIMEOUT: 5.0      # 最多旋转 5 秒
```

---

## 预期行为

1. **小车走到目标位置** 
2. **检查朝向是否满足**
   - 如果朝向误差 < 0.1 rad（5.7°）→ 停止
   - 如果朝向误差 ≥ 0.1 rad → 继续旋转
3. **最多旋转 5 秒**
   - 如果 5 秒内达到目标朝向 → 停止
   - 如果 5 秒后还未达到 → 强制停止（防止无限转）

## 如果小车还是不转，检查这些

### 1️⃣ 确认参数已加载
在小车启动后，查看日志中是否有这样的信息：
```bash
[robot_pid_local_planner] Entering yaw adjustment phase. target_yaw=..., current_yaw=...
```

如果没有看到，说明：
- ✅ 已编译新版本吗？`catkin_make --pkg robot_pid_local_planner -j4`
- ✅ 启动脚本后完全停止并重新启动？（不是 Ctrl+C）

### 2️⃣ 检查 RViz 导航目标设置
- 点击 **"2D Nav Goal"** 按钮
- **先点击位置**（点一下）
- **再拖动鼠标**（拖到想要的朝向方向，箭头会随着鼠标转动）
- 松开鼠标

**注意**：如果只是点击，没有拖动，RViz 可能默认设置朝向为 0，这可能是期望的朝向。

### 3️⃣ 查看日志信息
启用 DEBUG 日志来看朝向控制的细节：
```bash
# 在运行节点的终端中检查是否有类似日志
[robot_pid_local_planner] Adjusting yaw: e=..., wz=..., time=.../5.0
```

如果看到 `wz` 不为 0，说明正在旋转。

### 4️⃣ 检查超时时间
如果总是在 5 秒后停止，可能是朝向目标实在无法达到。尝试：
- 增加朝向容差：`YAW_TOL: 0.15`（从 0.1 改为 0.15）
- 增加超时时间：`YAW_ADJUST_TIMEOUT: 10.0`（从 5.0 改为 10.0）

## 常见情况

### 场景 A：小车走到目标位置，停了下来（没有转）
**原因**：朝向目标可能已经满足了
**检查**：RViz 中看小车箭头方向，和你指定的朝向是否已经很接近？

### 场景 B：小车走到目标位置，开始转动，然后停了（转动了一会儿）
**原因**：朝向达到了目标！🎉
**验证**：小车方向是否和你期望的一致？

### 场景 C：小车走到目标位置，开始转动，但转得很慢
**原因**：朝向误差很小，或 PID 参数不够高
**调整**：
```yaml
KP_YAW: 2.0  # 从 1.2 改为 2.0，加快旋转
MAX_WZ: 1.0  # 从 0.8 改为 1.0，允许更快的角速度
```

### 场景 D：小车转了 5 秒就停了，但还没转到位
**原因**：朝向目标太难达到（可能是系统延迟或漂移）
**调整**：
```yaml
YAW_TOL: 0.15       # 从 0.1 改为 0.15，增加容差（允许 8.6° 误差）
YAW_ADJUST_TIMEOUT: 10.0  # 从 5.0 改为 10.0，更长时间
```

---

## 参数调整建议

| 问题 | 参数 | 调整 |
|------|------|------|
| 小车转得太慢 | `KP_YAW` | 1.2 → 2.0 |
| 小车转得太快/抖动 | `KP_YAW` | 1.2 → 0.8 |
| 朝向容差不好满足 | `YAW_TOL` | 0.1 → 0.15 |
| 朝向容差要求太高 | `YAW_TOL` | 0.1 → 0.05 |
| 总是超时停止 | `YAW_ADJUST_TIMEOUT` | 5.0 → 10.0 |
| 朝向控制禁用 | `ENABLE_FINAL_YAW` | false → true |

## 完整工作流

```
┌─ 启动小车
│  roslaunch robot_pid_local_planner Omnidirectional_PID.launch
│
├─ 在 RViz 中设置目标
│  1. 点击 "2D Nav Goal"
│  2. 点击目标位置
│  3. 拖动鼠标设置朝向
│  4. 松开鼠标
│
├─ 小车执行导航
│  走向目标位置
│
├─ 位置到达后
│  检查朝向
│
└─ 朝向调整阶段（NEW）
   ├─ 朝向误差 < YAW_TOL（0.1 rad）→ 停止
   └─ 朝向误差 ≥ YAW_TOL → 旋转
      └─ 旋转时间 > YAW_ADJUST_TIMEOUT（5 秒）→ 强制停止
```
